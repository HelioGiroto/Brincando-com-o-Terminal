# PYAUTOGUI PARA INSTAGRAM

Teclas de atalho para uso do pyautogui para manipular Instagram pelo navegador (Firefox for Linux)

## Fazer Login na conta do Instagram:

	CONTROL + L	- No Firefox, posiciona para a barra de endereços (urls)
	URL da @conta	- Digita url da conta do Instagram
	ENTER		

	F5		- Refresca página

	TAB * 11	- Posiciona Mais (Botão hamburguer)
	ENTER

	TAB * 5		- Posiciona "Trocar de Conta"
	ENTER

	SHIFT TAB	- Posiciona "Entrar em uma conta já existente)
	ENTER

	TAB		- Posiciona input de nome da conta
	Digita @conta	

	SETA BAIXO	- Posiciona no nome da conta
	ENTER

	SHIFT TAB * 3	- Posiciona no botão ENTRAR
	ENTER



## Publicar Post (imagem + descrição):

	F5		- Refresca página

	TAB * 8		- Posiciona em "Criar"
	ENTER		- 

	TAB		- Seleciona "Postar"
	ENTER

	TAB		- Seleciona "Botão azul"
	ENTER 		- 

	Nome do arquivo	- Escolhe o nome da imagem para postar
	ENTER

	TAB * 3		- Para Seleciona Tamanhos
	ENTER

	SHIFT TAB * 2	- Para Seleciona Tam 4:5
	ENTER

	SHIFT TAB * 3	- Seleciona Avançar
	ENTER

	TAB * 2		- Seleciona Avançar
	ENTER

	TAB * 5		- Seleciona campo de Descrição
	CONTROL + V 	- Cola Texto de Descrição

	SHIFT TAB * 3	- Seleciona Compartilhar
	ENTER

		**Publicado**

	F5		- Refresca página
