# Os principais atalhos de movimentação no terminal Linux (Bash)

`Ctrl+A` (início da linha), `Ctrl+E` (final da linha), `Alt+F` (frente uma palavra) e `Alt+B` (trás uma palavra). Use setas para navegação simples, `Ctrl+L` para limpar a tela e `Ctrl+Shift+PageUp/Down` para rolar o histórico do terminal. 

Aqui estão os atalhos essenciais organizados por função: 

## Movimentação do Cursor 

- Ctrl + A: Move o cursor para o início da linha.
- Ctrl + E: Move o cursor para o final da linha. (*E*nd)

- Ctrl + Setas (Esquerda/Direita): Movem o cursor uma palavra de cada vez.
- Alt + F: Move o cursor uma palavra para *F*rente.
- Alt + B: Move o cursor uma palavra para trás. (*B*ack)

- Setas (Esquerda/Direita): Movem o cursor um caractere.
- Ctrl + F: Move o cursor um caractere para *F*rente.
- Ctrl + B: Move o cursor um caractere para trás. (*B*ack)


## Histórico e Tela 

- Ctrl + L: Limpa a tela (equivale ao comando clear).
- Shift + PageUp/PageDown: Rola a tela do terminal para cima ou para baixo.
- Seta para Cima/Baixo: Navega pelos comandos anteriores/próximos.
- Ctrl + R: Pesquisa reversa no histórico de comandos.

## Edição e Edição 

- Ctrl + U: Apaga do cursor até o início da linha.
- Ctrl + K: Apaga do cursor até o final da linha.
- Ctrl + W: Apaga a palavra antes do cursor.
- Ctrl + Y: Cola o texto que foi apagado com Ctrl+U/K/W.
