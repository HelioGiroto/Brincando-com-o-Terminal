                              Tman 6 years ago

                              Ok, fixed, after spending another 5 hours on it.

                              This works for Gnome and Cinnamon. I'm using Mint 17 Cinnamon.

                              .conkyrc file, needs to be edited to this:

                              own_window yes
                              own_window_transparent yes
                              own_window_type dock
                              own_window_hints undecorated,below,sticky,skip_taskbar,skip_pager
                              own_window_argb_visual yes
                              own_window_argb_value 100
                              gap_x 940
                              gap_y 20

                              I did not change gap_x or gap_y, as I was fine where it was, up to you. Suggest keeping an original .conkyrc file somewhere, as cya policy.

                              I was using Libreoffice. Make the changes, save( save as text), close, restart, should be working for you as it did me. Hope this helps someone else out there! :)

                              ...on that note...

                              Thank you so much on this site for all the help I received here.

                              Really makes a person want to join. ;)
